# My Portfolio Website
 
A personal portfolio website built using HTML and CSS.
 
## Live Site
https://sumadhur66-dev.github.io/my-website
 
## Sections
- **Home** – Hero introduction
- **About** – Background and details
- **Skills** – Technical skills
- **Projects** – Work and projects
- **Contact** – Get in touch
 
## Tech Used
- HTML5
- CSS3 (Flexbox, Grid, Transitions, Media Queries)
